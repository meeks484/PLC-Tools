Integer Array And Bit Number Conversions

Example:

  • The PLC is communicating to a FANUC Robot via Ethernet/IP.
  • The PLC ethernet module has its I/O setup as: SINT[25]
  • The Robot's DI/DO is set to 1 to 200.
  • You need to know what bit on the PLC is shared with the 142nd DI on the robot.
  • Get the array index and bit index by selecting 8 bit, and then enter bit 141.
  • The resulting integer index is 17, the bit index is 5.
