User Defined Datatype
Bit Stream

Copy and paste your UDT to a .CSV file, and then upload it here.

  • The first column needs to be the name of the tag
  • The second column needs to be your data type
  • All other data in the file will be ignored
  • Use brackets to indicate arrays (eg: BOOL[5])
  • STRING types are 88 BYTES unless its length is defined (eg: STRING5)
  • Replace custom data type names with "customT", and then followed by the number of BYTES in that type (eg: customT12)
  • File format exported from Studio Logix will work as is
  • Accepted Types: BOOL, SINT, USINT, INT, UINT, DINT, UDINT, REAL, STRING
